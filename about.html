About me

I'm a passionate and detail-oriented 17-year-old Full-Stack Developer with a strong foundation in both front-end and back-end technologies. On the front end, I specialize in building responsive, dynamic user interfaces using HTML5, CSS3, JavaScript, and React.js. My designs prioritize user experience, accessibility, and performance. On the back end, I’m proficient in Java for developing robust server-side applications, and I use MySQL for efficient data management and database architecture. I enjoy solving real-world problems through clean, scalable code and collaborating on meaningful projects. Whether it’s designing an intuitive user interface or building a secure backend system, I bring a balanced approach to every project. I’m always eager to learn, grow, and take on new challenges that push my skills further.
